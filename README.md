# Haskell_SoSe21

## to run application:
    1. open cmd
    2. go to fileReader.hs directory
    3. run runghc .\fileReader.hs 
    4. choose what you want to do
    5. repeat 3.-5. to check everything out
    6. Poems Database is "Gedichte3.json"
